# Rock Paper Scissors – Flask Web App

A simple web-based Rock Paper Scissors game built using Flask.  
The player competes against the computer through a browser interface.

## Features
- Play Rock, Paper, Scissors against the computer
- Random computer move generation
- Clean web interface using HTML, CSS, and JavaScript
- Tracks results during gameplay

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript

## How to Run the Project

1. Make sure Python is installed
2. Install Flask

```
pip install flask
```

3. Run the application

```
python app.py
```

4. Open your browser and go to:

```
http://127.0.0.1:5000
```
